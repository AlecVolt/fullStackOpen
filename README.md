## Full Stack Open Projects

This repository contains projects completed during the **Full Stack Open** course. Each section covers different aspects of modern web development, from the fundamentals of React to backend development, databases, authentication, testing, and deployment.

### Course Content

**Part 0 – Fundamentals of Web Apps**
HTTP, browsers, servers, and the basics of web application architecture.

**Part 1 – Introduction to React**
React components, JSX, props, state, event handling, and conditional rendering.

**Part 2 – Communicating with Server**
Forms, REST APIs, data fetching with Axios, and managing application state.

**Part 3 – Programming a Server with Node.js and Express**
Building REST APIs with Express, MongoDB integration, validation, and deployment.

**Part 4 – Testing Express Servers and User Administration**
Unit and integration testing with Jest, authentication using JSON Web Tokens (JWT), and user management.

**Part 5 – Testing React Apps**
React Testing Library, end-to-end testing with Playwright, and token-based authentication.

**Part 6 – Advanced State Management**
Redux Toolkit, React Query, Context API, and managing complex application state.

**Part 7 – React Router, Custom Hooks, and Styling**
Routing with React Router, custom hooks, code splitting, and advanced React patterns.

**Part 8 – GraphQL**
Building GraphQL APIs with Apollo Server and consuming them with Apollo Client.

**Part 9 – TypeScript**
Using TypeScript in React and Node.js applications, type-safe development, and best practices.
